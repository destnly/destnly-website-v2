
# Destnly Website

Destnly is a modern, visually engaging static website for a dating platform. It is designed to help people find meaningful connections in a safe and welcoming environment. The site features:

- Responsive design for mobile and desktop
- Smooth animations and interactive UI
- Informative pages: Home, Features, About, Help, Privacy Policy, Terms of Service, and Safety
- Clean, accessible layout and navigation
- No backend or database required (static HTML, CSS, JS)

## Getting Started

You can run this website locally or deploy it to any static hosting provider (GitHub Pages, Netlify, Vercel, etc.).

### To run locally:

1. **Clone or download this repository.**
2. Open the project folder in your file explorer.
3. Double-click `index.html` to open it in your browser.
	- Or, right-click and choose "Open with" your preferred browser.

#### Optional: Run with a local server (for best experience)

If you want to avoid CORS issues or use browser navigation features, you can run a simple local server:

**Using Python 3:**

```sh
python3 -m http.server 8000
```
Then visit [http://localhost:8000](http://localhost:8000) in your browser.

**Using Node.js (http-server):**

```sh
npm install -g http-server
http-server
```

## Project Structure

- `index.html` – Home page
- `features.html`, `about.html`, `help.html`, `privacy.html`, `terms.html`, `safety.html` – Informational pages
- `styles.css` – Main stylesheet
- `script.js` – Main JavaScript for UI/animations
- `assets/` – Images and additional CSS (e.g., animations)
